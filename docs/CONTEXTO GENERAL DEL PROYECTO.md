# PROJECT CONTEXT

Nombre del proyecto:
Encuentra Tu Hogar

Tipo de proyecto:
Plataforma web/app de arriendos comunitarios y seguros.

Problema principal:
La gentrificación y el aumento de precios de vivienda están desplazando
a habitantes locales de Medellín y otras ciudades.

Las plataformas tradicionales:
- generan altos costos,
- crean desconfianza,
- priorizan turismo temporal,
- aumentan desplazamiento económico,
- y reducen el acceso a vivienda estable.

Objetivo principal:
Crear una plataforma digital inteligente que conecte directamente
arrendadores y arrendatarios de forma segura, humana y comunitaria,
priorizando habitantes locales y reduciendo intermediarios.

La plataforma NO busca ser otro Airbnb.
Busca:
- estabilidad,
- comunidad,
- permanencia local,
- confianza,
- conexión humana,
- seguridad,
- y acceso justo a vivienda.

Diferenciadores:
- Prioridad a habitantes locales
- Comunidad y apoyo social
- Verificación de identidad
- Comunicación directa
- Menos intermediarios
- Seguridad de datos
- Transparencia
- Herramientas comunitarias
- Enfoque social y urbano

La aplicación ya tiene diseño UI definido.
NO se debe cambiar la identidad visual.
La arquitectura y funcionalidades deben adaptarse al diseño actual.
