# USER ROLES

## Arrendatario
Puede:
- buscar viviendas
- filtrar propiedades
- guardar favoritos
- contactar propietarios
- agendar visitas
- participar en comunidad
- dejar reseñas

## Arrendador
Puede:
- publicar viviendas
- editar propiedades
- responder solicitudes
- aceptar/rechazar interesados
- gestionar disponibilidad

## Administrador
Puede:
- moderar contenido
- validar usuarios
- gestionar reportes
- administrar publicaciones
- supervisar seguridad
