# UX/UI REQUIREMENTS

NO cambiar el diseño actual de la aplicación.

Mantener:
- identidad visual
- colores
- estructura base
- enfoque moderno y humano

La experiencia debe sentirse:
- segura
- amigable
- comunitaria
- moderna
- accesible
- emocionalmente cercana

NO parecer:
- Airbnb
- Booking
- marketplace genérico
